    **globals.css** -- Add three things after the existing `.grain-overlay` block:

    1. A global `*:focus-visible` rule with `outline: 2px solid var(--color-terracotta); outline-offset: 2px;`. This uses `outline` (not box-shadow) because outline is preserved in Windows High Contrast Mode. Then add `:focus:not(:focus-visible) { outline: none; }` to suppress outlines on mouse click.

    2. A `@media (prefers-reduced-motion: reduce)` block that sets `html { scroll-behavior: auto !important; }` and `.animate-bounce { animation: none; }` -- this covers the ScrollIndicator CSS bounce and any smooth scroll behavior.

    3. In the `@theme` block, add `--color-terracotta-dark: #a85e30;` after the existing `--color-terracotta-light` line. This darker terracotta achieves ~5.0:1 contrast on cream and ~5.7:1 on white, meeting WCAG AA for normal text.

    **MotionProvider.tsx** -- Import `MotionConfig` from `motion/react`. Wrap the existing `<LazyMotion>` inside `<MotionConfig reducedMotion="user">`. This automatically disables transform animations (y, x, scale, rotate) on all `m.*` components while preserving opacity fades. The structure should be: `MotionConfig > LazyMotion > children`.
  </action>
