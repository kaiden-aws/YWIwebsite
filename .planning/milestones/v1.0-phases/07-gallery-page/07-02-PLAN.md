**GalleryLightbox.tsx** ŌĆö No `'use client'` directive needed (it is only imported by the client component GalleryGrid):
- Import `useEffect`, `useRef`, `useCallback` from `react`
- Import `AnimatePresence`, `m` from `motion/react`
- Import `X`, `ChevronLeft`, `ChevronRight` from `lucide-react`
- Import `ImagePlaceholder` from `@/components/ui/ImagePlaceholder`
- Import `type Project` from `@/lib/data/projects`

Props interface `GalleryLightboxProps`:
- `images: Project[]` ŌĆö the currently filtered project array
- `selectedIndex: number | null` ŌĆö null when closed
- `onClose: () => void`
- `onPrev: () => void`
- `onNext: () => void`

Refs:
- `modalRef = useRef<HTMLDivElement>(null)` ŌĆö for focus trap querySelectorAll
- `previousFocusRef = useRef<HTMLElement | null>(null)` ŌĆö restore focus on close

**Keyboard handler** (`useEffect`):
- Only attach when `selectedIndex !== null`
- `keydown` event listener on `document`:
  - `Escape` ŌåÆ `onClose()`
  - `ArrowLeft` ŌåÆ `onPrev()`
  - `ArrowRight` ŌåÆ `onNext()`
  - `Tab` ŌåÆ Focus trap logic:
    - Query `button, [tabindex]:not([tabindex="-1"])` inside `modalRef.current`
    - If `shiftKey` and active element is first focusable ŌåÆ `preventDefault()` and focus last
    - If not `shiftKey` and active element is last focusable ŌåÆ `preventDefault()` and focus first
- Cleanup: `removeEventListener`

**Body scroll lock + focus management** (`useEffect`):
- When `selectedIndex !== null`:
  - Store `document.activeElement` in `previousFocusRef`
  - Set `document.body.style.overflow = 'hidden'`
  - `requestAnimationFrame(() => modalRef.current?.focus())` to focus modal after animation
- When `selectedIndex` becomes null:
  - Restore `document.body.style.overflow = ''`
  - Restore focus: `previousFocusRef.current?.focus()`
- Cleanup function: `document.body.style.overflow = ''` (safety net for unmount)

**Modal rendering** (wrapped in `AnimatePresence`):
- Only render when `current` exists and `selectedIndex !== null`
- Outer `m.div` (the modal):
  - `ref={modalRef}`, `tabIndex={-1}`, `role="dialog"`, `aria-modal="true"`
  - `aria-label={\`${current.name} ŌĆö ${current.category}\`}`
  - `className="fixed inset-0 z-50 flex items-center justify-center"`
  - `initial={{ opacity: 0 }}`, `animate={{ opacity: 1 }}`, `exit={{ opacity: 0 }}`

- Backdrop: `div` with `absolute inset-0 bg-charcoal/90`, `onClick={onClose}`

- Content `m.div`:
  - `className="relative z-10 w-full max-w-5xl mx-4"`
  - `initial={{ scale: 0.9, opacity: 0 }}`, `animate={{ scale: 1, opacity: 1 }}`, `exit={{ scale: 0.9, opacity: 0 }}`
  - `transition={{ duration: 0.2 }}`

- Close button: positioned `absolute -top-12 right-0`, `aria-label="Close lightbox"`, uses `X` icon (h-6 w-6), `text-cream/80 hover:text-cream p-2`

- Image: `ImagePlaceholder` with `label={current.image}`, `aspectRatio="video"`, `className="!rounded-lg w-full"`

- Caption below image: `mt-4 text-center`
  - Project name: `text-cream font-display text-xl`
  - Category: `text-cream/60 text-sm`
  - Counter: `text-cream/40 text-xs mt-1` showing `{selectedIndex + 1} / {images.length}`

- Prev button: `absolute left-0 top-1/2 -translate-y-1/2 -translate-x-12 md:-translate-x-14`, `aria-label="Previous image"`, `ChevronLeft` icon (h-8 w-8), `text-cream/80 hover:text-cream p-2`
- Next button: `absolute right-0 top-1/2 -translate-y-1/2 translate-x-12 md:translate-x-14`, `aria-label="Next image"`, `ChevronRight` icon (h-8 w-8), `text-cream/80 hover:text-cream p-2`

Navigation wraps circularly (handled by parent via modular arithmetic in onPrev/onNext).
