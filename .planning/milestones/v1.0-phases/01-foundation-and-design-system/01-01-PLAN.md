    Note on FNDN-09 (spacing): Per research recommendation, keep Tailwind's default `--spacing: 0.25rem` (4px base). The "generous whitespace with 8px base scale" requirement is achieved by convention — using larger spacing values like `py-16` (64px) for sections, `gap-8` (32px) for grids, `space-y-6` (24px) for content flow. Overriding `--spacing` to `0.5rem` would double every utility and confuse any developer familiar with standard Tailwind. The spacing CONVENTION is documented via the test page in Plan 01-02.

    Create `src/lib/utils/cn.ts`:

    ```typescript
    import { type ClassValue, clsx } from 'clsx'
    import { twMerge } from 'tailwind-merge'

    export function cn(...inputs: ClassValue[]) {
      return twMerge(clsx(inputs))
    }
    ```
