    **Create `src/components/layout/MobileDrawer.tsx`** ('use client'):

    A slide-in navigation drawer for mobile using AnimatePresence.

    Props: `isOpen: boolean`, `onClose: () => void`, `children: React.ReactNode`.

    Implementation:
    - Import `AnimatePresence` and `m` from `motion/react`, `X` from `lucide-react`
    - Wrap everything in `<AnimatePresence>`. Only render children when `isOpen` is true.
    - Render two direct children of AnimatePresence (both conditionally rendered inside `{isOpen && (<>...</>)}`):
      1. **Backdrop overlay**: `m.div` with `key="backdrop"`, initial opacity 0, animate opacity 1, exit opacity 0, duration 0.2. Classes: `fixed inset-0 z-40 bg-charcoal/50`. `onClick={onClose}`, `aria-hidden="true"`.
      2. **Drawer panel**: `m.div` with `key="drawer"`, initial `{ x: '100%' }`, animate `{ x: 0 }`, exit `{ x: '100%' }`, transition `{ type: 'tween', duration: 0.3, ease: [0.22, 1, 0.36, 1] }`. Classes: `fixed right-0 top-0 z-50 h-full w-[280px] bg-cream shadow-xl`.
      3. Inside the drawer panel: a close button (`<button onClick={onClose} aria-label="Close menu">` with `X` icon, positioned `absolute right-4 top-4 p-2`, icon `h-6 w-6 text-charcoal`). Then a `<nav className="mt-16 px-6">` wrapping `{children}`.
    - CRITICAL: The `{isOpen && (<>...</>)}` fragment must be a direct child of `AnimatePresence` for exit animations to work. The backdrop and drawer m.divs must have unique `key` props.

    **Create `src/components/layout/HeaderClient.tsx`** ('use client'):

    This component owns ALL interactive header behavior: scroll detection, active link highlighting, mobile drawer toggle, and renders the full header markup.

    Props: `children?: React.ReactNode` (for any server-rendered content passed from Header.tsx, though may not be needed).

    Implementation:
    1. **Imports**: `useState`, `useEffect` from react. `usePathname` from `next/navigation`. `Link` from `next/link`. `m` from `motion/react`. `Menu`, `Phone` from `lucide-react`. `cn` from `@/lib/utils/cn`. `navLinks`, `companyInfo` from `@/lib/data/navigation`. `MobileDrawer` from `./MobileDrawer`.

    2. **Scroll detection**: Use a `useScrolled` custom hook (can be inline or a named function within the file). `useState(false)` for `scrolled`. `useEffect` adds a scroll listener with `{ passive: true }` that sets `scrolled` to `window.scrollY > 50`. Call handler once on mount to check initial position. Return cleanup.

    3. **Drawer state**: `useState(false)` for `isOpen`. Add a `useEffect` watching `pathname` (from `usePathname()`) that sets `isOpen(false)` on change -- this auto-closes the drawer when navigating.

    4. **NavLink sub-component** (inline in the file): Accepts `href`, `children`, `className?`, `onClick?`. Uses `usePathname()` to determine `isActive = pathname === href`. Renders `<Link>` with:
       - Desktop active style: `text-cream` + an `after:` pseudo-element underline in terracotta (`after:absolute after:bottom-0 after:left-0 after:right-0 after:h-0.5 after:bg-terracotta after:content-['']`)
       - Desktop inactive style: `text-cream/70 hover:text-cream`
       - Apply `relative py-2 text-sm font-medium transition-colors` as base classes
       - Use `cn()` to merge

    5. **MobileNavLink sub-component** (inline): Similar to NavLink but with mobile-specific active style: `bg-forest/10 text-forest font-semibold` when active, `text-charcoal hover:text-forest` when inactive. Renders as block-level links with `py-3 px-4 rounded-lg text-base transition-colors`. Accepts an `onClick` prop that calls `onClose` to dismiss drawer on tap.

    6. **Render structure**:
       ```
       <header className={cn(
         'sticky top-0 z-50 transition-all duration-300',
         scrolled ? 'bg-forest/85 backdrop-blur-md shadow-sm' : 'bg-forest'
       )}>
         <div className="mx-auto max-w-7xl px-4 sm:px-6">
           <div className="flex h-16 items-center justify-between md:h-20">

             {/* Logo — left side */}
             <Link href="/" className="flex items-center gap-2">
               <span className="font-display text-xl md:text-2xl text-cream">
                 YARD WEASELS
               </span>
             </Link>

             {/* Desktop nav — center, hidden on mobile */}
             <nav className="hidden md:flex items-center gap-6">
               {navLinks.map(link => <NavLink key={link.href} href={link.href}>{link.label}</NavLink>)}
             </nav>

             {/* Right side: phone + CTA (desktop) + hamburger (mobile) */}
             <div className="flex items-center gap-3">
               {/* Phone — desktop only */}
               <a href={companyInfo.phoneHref} className="hidden lg:flex items-center gap-1.5 text-sm text-cream/80 hover:text-cream transition-colors">
                 <Phone className="h-4 w-4" />
                 {companyInfo.phone}
               </a>

               {/* CTA button — visible at all breakpoints */}
               <Link href="/contact" className="bg-terracotta hover:bg-terracotta-light text-cream text-sm font-medium px-4 py-2 rounded-lg transition-colors">
                 Get a Quote
               </Link>

               {/* Hamburger — mobile only */}
               <button
                 onClick={() => setIsOpen(true)}
                 className="md:hidden p-2 text-cream"
                 aria-label="Open menu"
               >
                 <Menu className="h-6 w-6" />
               </button>
             </div>
           </div>
         </div>
       </header>

       {/* Mobile drawer */}
       <MobileDrawer isOpen={isOpen} onClose={() => setIsOpen(false)}>
         <div className="flex flex-col gap-1">
           {navLinks.map(link => (
             <MobileNavLink key={link.href} href={link.href} onClick={() => setIsOpen(false)}>
               {link.label}
             </MobileNavLink>
           ))}
         </div>
         {/* Phone number in drawer for mobile */}
         <a href={companyInfo.phoneHref} className="mt-6 flex items-center gap-2 text-charcoal py-3 px-4">
           <Phone className="h-5 w-5 text-forest" />
           {companyInfo.phone}
         </a>
       </MobileDrawer>
       ```
