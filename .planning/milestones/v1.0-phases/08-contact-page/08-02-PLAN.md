**Form State (useState with string values):**
```typescript
interface ContactFormState {
  name: string
  email: string
  phone: string
  serviceInterest: string
  message: string
}
interface ContactFormErrors {
  name?: string
  email?: string
  message?: string
}
```
Initialize: `{ name: '', email: '', phone: '', serviceInterest: '', message: '' }`
Also: `const [errors, setErrors] = useState<ContactFormErrors>({})` and `const [submitted, setSubmitted] = useState(false)`

**Validation function (validate-on-submit only, NOT on keystroke):**
- Name: required, trim check → "Name is required"
- Email: required + regex `/^[^\s@]+@[^\s@]+\.[^\s@]+$/` → "Email is required" or "Please enter a valid email address"
- Message: required, trim check → "Message is required"
- Phone: optional (no validation)
- Service Interest: optional (no validation — empty string is fine)

**Form handler:**
```typescript
function handleSubmit(e: React.FormEvent) {
  e.preventDefault()
  const newErrors = validate()
  setErrors(newErrors)
  if (Object.keys(newErrors).length === 0) {
    setSubmitted(true)
  }
  // NEVER touch form state on validation failure — preserves all input values
}
```
